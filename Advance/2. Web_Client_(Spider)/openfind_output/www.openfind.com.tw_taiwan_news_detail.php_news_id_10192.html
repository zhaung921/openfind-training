Openfind 攜手是方電訊、群環科技、關貿網路發表雲端資安生態圈聯盟，提供企業創新的全方位資安應用服務

【由左至右為是方電訊總經理劉耀元，網擎資訊執行長廖長健，群環科技雲端事業部副總胡世衛，關貿總經理張陸生】

MIC 2019 全球雲端產業報告指出，近年運用 AI 人工智慧、物聯網與區塊鏈等新興科技進行數位轉型的發展與日俱增，促使企業加速使用雲端運算服務，加上進階資安威脅挑戰日新月異，因此 Openfind 日前正式與合作夥伴共同提出了雲端資安生態圈聯盟 (CSEA，Cloud Security Ecosystem Alliance ) 的概念，宣示 Openfind 將從雲端智慧資安服務出發，透過軟體開發與 AI 應用，結合技術、維護與服務資源的全新價值主張，提供企業數位轉型與多雲整合的資安創新服務，打造最完整的雲端資安防護願景。

全球資安問題層出不窮，企業面臨縱深防禦新挑戰

眼看雲端趨勢銳不可擋，加上 2018 年歐盟更推出史上最嚴格的個資保護規範 GDPR，隨著層出不窮的跨國商業電郵詐騙 BEC (Business Email Compromise) 頻傳，乃至轟動一時的勒索病毒 WannaCry 肆虐，陸續發生的重大資安威脅事件，都讓我們深刻體認全面提升資安等級刻不容緩！有鑑於企業客戶無不期盼在躍上雲端之際，能有一個安心安全的雲端防護夥伴同行，Openfind 推出功能 All-in-one 的嶄新 Security Suite 雲端安全服務 OSecure (Openfind Secure Cloud Service)，聚焦在智慧溝通、資安防護及雲端安全等三大方向，涵蓋了內寄防護、外寄稽核、歸檔備份、檔案安全、顧問諮詢與技術服務 … 等多項雲端資安服務內容，提供企業用戶一站購足、彈性組合的便利性，其不限自建、公有雲、私有雲、混合雲架構，不限郵件主機品牌皆可搭配的彈性，不但可強化組織內部溝通安全，更是協助企業快速提升資安防禦等級的最佳選擇。

結合 AI 智慧創新應用，跨雲整合提供一致性解決方案

根據市場調查機構 IDC 2019 年台灣市場十大 ICT 預測報告中指出，預期在 2023 年時，人工智慧將成為協助企業轉型的重要技術。網擎資訊執行長廖長健表示：「台灣企業面臨數位轉型與進階資安威脅挑戰，無論是從多雲管理  (Multi-Cloud) 或使用者為中心 (User-centered) 的雲端服務架構，皆需要規劃資

訊安全與多雲整合的策略方法。」「雲端資安生態圈聯盟 (CSEA，Cloud Security Ecosystem Alliance ) 由網擎資訊發起，以雲端服務為中心，希冀透過軟體開發與 AI 智慧創新應用，加速整合是方電訊、群環科技與關貿網路的多雲服務，藉由技術、服務、銷售等一致性的資源方案提供，協助企業敏捷回應，提升混合與多雲服務競爭力。」廖長健指出。

關於是方電訊

是方電訊主要提供資通訊整合與ICT應用服務，其中是方應用雲App Cloud平台，以雲服務供應商 (SaaS Vendor) 出發，提供台灣雲端業者一個市集平台 (Cloud Marketplace) 通路，服務種類涵蓋虛擬實境與擴增實境、企業商務運用、資安認證、商務行銷與產業應用等五大領域。協助合作夥伴推廣雲端服務，讓台灣中小企業能夠享受到兼具安全、高品質的雲端服務。更多訊息，請瀏覽是方應用雲https://www.chief.com.tw/dispPageBox/ct.aspx?ddsPageID=CHAPPLICATION&dbid=4616136061

關於群環科技

群環科技以雲端多元應用與專業加值服務為主，發展 BestCloud 雲端品牌服務，提供「備份備援、資訊安全、智慧辦公室、企業必備」的雲服務，結合多家雲端品牌的解決方案，運用群環在商用市場營運知識，發展應用整合與加值服務，以全方位的 BestCloud Solution 成為企業客戶最佳雲端解決方案夥伴。更多訊息，請瀏覽群環雲端 https://bestcloud.bestcom.com.tw/

關於關貿網路

關貿網路資訊安全監控中心 (SOC) 提供 7x24 全年無休的「資訊安全監控服務」。服務範圍涵蓋網路攻擊與入侵事件即時監控、即時通報、線上緊急應變處理及防護建議，提供企業高度安全的網路防護環境，免除資訊安全威脅之隱憂，讓企業安心致力於本身業務的開拓與發展，提昇整體的企業競爭力。更多訊息，請瀏覽關貿網路https://www.tradevan.com.tw/services/index.do?act=services_info&type=32

關於 Openfind Secure 雲端資安服務

Openfind Secure 雲端資安服務為網擎推出的雲端資安服務品牌，提供企業客戶最新型的資安縱深防禦服務，包含「收信安全、外寄安全、稽核歸檔、檔案保全、資安顧問與技術支援」等服務，結合 AI 人工智慧技術，讓企業敏捷回應各種郵件資安威脅。更多訊息，請瀏覽 Openfind Secure 雲端資安服務 https://www.openfind.com.tw/osecure/

關於 Openfind

Openfind 成立於 1998 年 1 月，長期致力於網際網路相關技術研發，為企業用戶與廣大網友提供高品質、高穩定與高擴充性的軟體與服務。從企業市場搜尋引擎系列產品、電子郵件伺服器或郵件防護、稽核、歸檔系統，到一般使用者的電子郵件信箱與企業郵件代管服務，Openfind 堅守「創新技術與優質服務」的企業文化，以提供使用者最符合成本效益的解決方案。更多訊息，請瀏覽公司網站 http://www.openfind.com/
