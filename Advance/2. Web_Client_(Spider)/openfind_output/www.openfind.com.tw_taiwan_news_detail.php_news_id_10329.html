Openfind Mail2000 優化異常登入警示協助資安長跳脫駭客狙擊圈

隨著 2019 年資通安全管理法上路後，近年資安法規不斷優化，包含公務機關、上市、上櫃等公開發行的企業組織，都陸續有明文規定應遵循的資安執行框架。考量到資安事件已與公司營運密不可分，金管會更於2021年底明定公開發行之企業需設置資安長一職，職責包括監視和評估組織的資訊安全風險、研究和推薦安全技術，並與其他部門領導和IT部門合作來實施和維護安全措施。有鑑於此，Mail2000研發團隊進一步優化Mail2000 V8異常登入警示，協助資安長輕鬆即時監控、預防資安事件發生。

然而，在駭客攻擊手法日益精進，網路攻防越演越烈下，企業雖無法百分百避免，但仍可積極建立防禦機制，進而在第一時間確實掌握資安事件發生動態。因此，Mail2000原本可針對 IP 所在位置做偵測，若 IP 出現在不常見的區域，系統會主動通知，讓管理者立即查看並處理異常登入狀況或將 IP 設定於白名單，V8 異常登入警示最新版了透過原本 IP 偵測設定還可以設定帳號異常登入行為達到全面的即時監控及快速反應異常行為，更提供多種異常登入行為分析設定，讓管理者能依據公司狀況彈性調整，最後再搭配自動化的監控報告數據，大幅降低管理者負擔並提高用戶的體驗，同時達到企業員工安全意識的提升。

                      

【IP白名單設定】管理者可以針對特定 IP 設定白名單

 

                                                                

【設定異常登入條件】新版功能管理者可設定：一天內登入幾次、一天內匯出信件幾次等...

 

網擎資訊產品部門經理賴濬倩指出：「Mail2000 V8異常登入警示最新版，是研發團隊透過實際攻防的經驗及彙整十多家企業用戶需求，進而優化此項功能。資安長可彈性地依照公司情況設定異常登入規則並根據這些條件設定相對應的反應機制。再搭配即時監控帳號活動，並在異常情況發生時快速得知，以便立即採取適當的反應。」「此外，當IT管理者希望瞭解哪些帳號在過去一週有異常登入行為時，『查看異常登入的歷史紀錄』功能可以快速地提供答案，新功能為管理者提供了更多的情境工具，幫助他們在面對複雜的資訊安全挑戰時作出明智的決策。」賴經理指出。

新版本異常登入功能透過主動設定防護規則，找出有異常行為的帳號，幫助提升系統安全性。也能夠透過報表、歷史紀錄統計異常登入的行為，協助資安長更輕鬆的完成日常工作。Mail2000 V8可確保郵件在功能齊全又可靠的系統平台上平穩運作，加上今年6 月底，Mail2000 V7 已經停止維護，升級到最新版本不僅可以確保系統免受潛在的風險與威脅，還能體驗到更多新功能帶來的便利性。
