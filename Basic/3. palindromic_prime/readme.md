# Palindromic Prime (質數回文)
## 題目
使用者給定一個 N 值，列出 1 ~ N 裡所有的質數回文數。

質數回文數：該數字同時滿足下列兩個條件：
質數：除 1 與本身之外，沒有其他的因數。
	例如：2, 3
回文：從左向右和從右向左讀其結果值相同。
	例如：1, 22, 5665
N 值最大為 100,000,000

輸入：程式透過命令參數列來取得到 N 值
輸出：符合的數值以及個數
```
Eg.
./palprime 10
2 3 5 7  
count: 4  
```

N 值為 10
符合的數值為 2, 3, 5, 7，個數為 4 個

請思考能夠有能夠有最佳 Performance 的做法。
