Openfind 部署後量子加密技術，強化郵件安全，保護政府與企業敏感資料

隨著量子電腦技術的迅速發展，我們即將邁入嶄新的後量子（Post-Quantum）時代。在這個時代中，傳統的加密演算法將不再安全，後量子加密技術（Post-Quantum Cryptography，簡稱PQC）即為了對抗此類威脅而設計的。包含美國國家標準與技術研究院（NIST）及國內學者專家都正積極應對，而面對這樣的全球威脅， Google 和 Cloudflare 等大型科技公司都已採取行動提供相關功能。

根據專家預估在 2029 年左右，量子電腦將能被大眾使用，並能解密現在幾乎所有的傳統加密數據。而現在已有許多企業組織擔心「先竊取，後解密（Harvest Now, Decrypt Later，簡稱 HNDL）」的攻擊手法，有心人士只要先收集加密資訊，未來一旦取得量子電腦運算能力即可馬上獲取其中的機敏資訊。因為 Email 傳輸中常包含大量的敏感和機密資訊，如果不立即加以保護，後果將不堪設想。

有鑑於此，身為郵件資安重要台灣原廠，Openfind 網擎資訊已率先開發部署相關防護，於系統所有對外傳輸層中，使用能抗量子破解的加密技術，支援 X25519Kyber768 產生 TLS 加密連線中的對稱式金鑰。網擎資訊執行長廖長健指出 ： 「只要使用 Google Chrome 瀏覽器 116 版本之後內建的加密技術，導入已具有 PQC 技術的 Openfind Mail2000 主機，所有的 Webmail 與 API 的傳輸數據，已具有抗量子攻擊的技術保護。」

Openfind 會繼續密切關注並遵循政府及 NIST 建議的後量子加密標準，並根據最新的標準與技術，持續提供全面的資訊安全解決方案。不論是預防量子攻擊，或各項強化客戶溝通安全的措施，網擎資訊都將不遺餘力地保護政府和企業客戶的郵件安全，確保珍貴的資料能在這個快速變化的數位時代中得到最可靠的保障。
