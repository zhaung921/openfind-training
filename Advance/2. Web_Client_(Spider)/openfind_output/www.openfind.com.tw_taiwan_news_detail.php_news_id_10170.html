駭客攻擊與資料外洩層出不窮，便利又安全的檔案分享在哪裡？

近年來金融資安事件層出不窮，2016 年第一銀行ATM遭駭盜領高達 8 千多萬，緊接著 2017 年遠東銀行遭駭客植入木馬盜轉 18 億，雖然後續都及時追回大部分款項，仍震驚國際。隨著雲端浪潮蔚為主流，企業內部檔案分享與傳輸安全的需求與日俱增，金融產業更首當其衝，去年以來數家銀行反映遭到主管機關查核內部使用 USB 傳輸檔案管控不當！正因為金融業對資訊安全的要求高，一旦使用者存取檔案及對外分享過程沒有完整記錄，管理者自然難以管控、更無法稽核到可能異常的檔案交換行為。

 

金融單位在掌握大量機敏及個人資料的情況下，檔案的分享需要更加嚴格規範，隨著主管機關下達禁用 USB 的要求，如何讓員工能簡便又安全地交換檔案，也成為金融業需要正視的挑戰。Openfind ArkEase Pro 雲端服務平台針對所有分享的檔案連結均能設定密碼與有效期限，同時也提供「檔案不落地」解決方案，透過完整的權限控管功能，除非有編輯或下載的權限，否則僅能預覽不能分享，同時檔案接收者也無法複製或列印；即使是用拍照的方式留存檔案，因系統已自動加上分享者帳號的浮水印，將可確保資料不被外流濫用。

 

Openfind 產品部門經理張弘達表示：「目前多數重要、機敏甚至客戶的個資都靠數位格式儲存，取用和傳輸很容易，對企業則是莫大的外洩風險。員工因工作需求必須共用或傳送檔案時，如何確保分享的過程安全無虞？」「以資安要求最嚴格的金融業為例，除了滿足使用者工作所需，更重要的是能確保檔案分享時的安全、以符合法規要求為最高原則。近期許多銀行、壽險業陸續導入內容協作平台 (Content collaboration platforms, CCP)，取代早期 FTP 或 USB 等方式，確保檔案在內外傳輸時能嚴格管控，即為最好的例子。」張經理表示。

 

ArkEase Pro 不但在安全層面做足全方位考量，更會以 Openfind 在郵件稽核的豐富經驗，持續強化 CCP 之稽核管理功能、滿足所有檔案分享過程中的細節要求。未來管理者將可稽核所有外寄檔案連結，即時掌控不當的分享行為、阻斷風險。透過保存所有的存取記錄，ArkEase Pro可完整稽核所有檔案操作；當疑似資安事件發生時能透過記錄查找問題，並在第一時間關閉所有異常分享，大幅降低資料外洩的威脅，協助企業打造便利使用又堅固安全的檔案分享平台！
